# 交易复盘分析仪表盘 (Trading Dashboard) - 产品白皮书与用户手册

## 1. 产品概述 (Product Overview)
**Trading Dashboard** 是一款专为加密货币交易员和量化分析师打造的个人交易日志与深度复盘平台。它旨在通过直接接入交易所 API（如 Binance, OKX 等），自动同步并归集海量原子成交记录，将其转化为结构化的“交易持仓 (Legs)”，并提供机构级的多维数据可视化分析。

核心理念是：**“不仅告诉您赚了多少，更要告诉您为什么赚，以及错在哪。”**

## 2. 核心功能模块 (Core Modules)

本系统由五个主要页面和一个全局框架构成，形成完整的复盘工作流：

### 2.1 全局界面框架 (Global UI)
*   **自适应侧边栏 (Sidebar)**：支持一键折叠/展开，为数据密集的图表腾出最大屏幕空间。包含各核心模块的快速导航。
*   **全局底部状态栏 (Bottom Bar)**：悬浮于全站底部，左侧实时显示用户的**当前净值 (Net Worth)**，右侧提供 Twitter, Discord, Telegram 等社区快捷入口，不干扰图表视线，极具专业交易终端质感。

### 2.2 仪表盘主页 (Dashboard - `/`)
*   **概览指标 (Overview Metrics)**：快速浏览总盈亏、总胜率、交易单数、期望值、平均盈亏、多空比率等 8 项核心数据。
*   **资金曲线 (Cumulative PNL Curve)**：通过大面积平滑折线图，直观展示最近 30 天的净值增长趋势。
*   **历史交易明细 (Trade History)**：包含分页功能的数据表（支持选择每页展示 10/20/50 条），展示每笔交易的开平仓价格、持仓时长及已实现盈亏，点击单行可弹出带进出场标记的 K 线图详情。

### 2.3 深入分析 (Analytics - `/analytics`)
针对高阶交易员的多维数据拆解中心，所有数据均支持通过右上角的**超级筛选器**（按账户、币种、时间范围）进行联动过滤：
*   **核心统计资料卡 (Statistics)**：详细列出净利润额、盈亏比、日均增益、平均单笔获利/亏损、最大连胜/连败次数，以及红色高亮的**重大损失 (Max Drawdown)**。
*   **多空方向表现拆解**：采用两个“甜甜圈”环形图，分别对比**多头获胜率**和**空头获胜率**，并对比两者的平均持仓时间。
*   **交易币种报告 (Symbol Report)**：列出各个交易对的具体表现，包含获胜百分比（带视觉进度条）、多空比、总增长等。
*   **四象限图表分析矩阵**：
    1.  **按天计算结果 (Weekday PnL)**：评估一周中哪天的胜率最高。
    2.  **按小时统计 (Hourly PnL)**：24小时资金波动曲线，寻找最佳开仓时段。
    3.  **交易持续时间报告 (Duration Report)**：直方图展示不同持仓时长（如 1-4 小时 vs 1 天以上）的盈利分布。
    4.  **交易规模报告 (Size Report)**：分析投入资金规模与最终盈亏的正负相关性。

### 2.4 交易日历 (Calendar - `/calendar`)
*   **月度热力图**：以日历的形式呈现每天的净盈亏。盈利日显示绿色金额，亏损日显示红色金额，辅以视觉进度条直观对比盈亏幅度。
*   **智能导航**：支持一键“回到今天”，或者快速跳转指定年份和月份。
*   **极值统计**：日历下方自动提炼当月的“最佳交易日”、“最差交易日”及“最大连胜天数”。

### 2.5 AI 交易报告 (AI Reports - `/reports`)
*   **一键诊断**：利用大语言模型（LLM），对用户的海量历史订单进行深度扫描。
*   **个性化复盘正文**：报告包含“表现亮点”（如：趋势跟随坚决）、“风险警告”（如：存在扛单倾向、频繁微小交易），并给出具体的“AI 核心建议”（如：停止交易特定币种、设置硬性止损线）。
*   **历史存档**：左侧边栏保存往期的周度、月度诊断报告供随时查阅。

### 2.6 账号配置 (Accounts - `/accounts`)
*   **多账户管理**：支持绑定多个不同平台的 API Key（如：Binance 主账户、OKX 冲锋号）。
*   **安全提示与状态**：强调 API 仅需“只读”权限。列表实时显示各个 API 密钥的连通状态（正常/失效）以及最后同步时间。

## 3. 技术架构 (Technical Architecture)

*   **前端框架**: Next.js 14+ (App Router), React, TypeScript.
*   **UI 样式**: Tailwind CSS (全局深色模式), Lucide React (图标).
*   **数据可视化**: ECharts (`echarts-for-react`) 用于统计图表，Lightweight Charts 用于高保真 K 线回放。
*   **后端支撑方案 (待接入)**: 
    *   **ORM / 数据库**: Prisma + PostgreSQL。
    *   **交易同步引擎**: `ccxt` (Node.js)，负责抹平各大交易所接口差异，并运用 FIFO（先进先出）算法将散乱的 Trade 拼接为带有精确盈亏和时间的 Leg。
    *   **API**: Serverless API Routes 提供高效的分页与 `GROUP BY` 聚合查询。

## 4. 产品愿景
Trading Dashboard 致力于将机构级的量化分析工具平民化，通过自动化数据清洗与 AI 深度介入，帮助每一位独立交易员克服人性弱点，建立稳定、可盈利的机械化交易系统。
